Keyboard Shortcuts

  • Enter: Add box below
  • Tab: Add box right
  • Ctrl + Backspace: Delete box
  • Alt + E: Move/Create right
  • Alt + Q: Move/Create left
  • Alt + W: Move/Create above
  • Alt + S: Move/Create below
  • Drag Box: Move box
